# Revenue Reports

This directory is the public reporting channel for AI Market Cap revenue transparency.

Each monthly report should be added as:

- `YYYY-MM.md`

Each report should include:
- gross platform revenue
- deductions
- net platform revenue
- treasury / operations / collaborator allocations
- collaborator points and payout breakdown
- maintainer sign-off

Use [`_template.md`](./_template.md) as the starting format.
