# Code Of Conduct

AI Market Cap is an open collaboration project. We expect contributors, reviewers, maintainers, and community members to act like professionals.

## Expected behavior

- be direct and respectful
- critique code and decisions, not people
- explain technical reasoning clearly
- keep discussions relevant to the project
- help maintain a high-signal working environment

## Unacceptable behavior

- harassment, intimidation, or abusive language
- discrimination or exclusionary behavior
- spam, trolling, or bad-faith debate
- deliberate misinformation about project behavior, governance, or revenue handling
- doxxing, threats, or impersonation

## Maintainer responsibility

Maintainers are expected to:
- enforce this code of conduct consistently
- keep moderation decisions grounded in project safety and quality
- document serious enforcement actions when appropriate

## Reporting

If you need to report a conduct issue, use:
- https://aimarketcap.tech/contact

Do not use public issues for sensitive interpersonal or safety reports.

## Enforcement

Depending on severity, maintainers may:
- request behavior changes
- lock or moderate a thread
- close a PR or issue
- suspend collaboration privileges
- remove access from repeat or severe offenders

This code of conduct exists to protect the project and the people building it.
