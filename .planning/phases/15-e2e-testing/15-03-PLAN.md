    e2e:
      name: E2E
      runs-on: ubuntu-latest
      timeout-minutes: 15
      env:
        NEXT_PUBLIC_SUPABASE_URL: ${{ secrets.NEXT_PUBLIC_SUPABASE_URL }}
        NEXT_PUBLIC_SUPABASE_ANON_KEY: ${{ secrets.NEXT_PUBLIC_SUPABASE_ANON_KEY }}
      steps:
        - uses: actions/checkout@v4
        - uses: actions/setup-node@v4
          with:
            node-version: 24
            cache: 'npm'
        - run: npm ci
        - name: Install Playwright Browsers
          run: npx playwright install --with-deps chromium firefox
        - name: Run E2E tests
          run: npx playwright test
        - uses: actions/upload-artifact@v4
          if: failure()
          with:
            name: playwright-report
            path: playwright-report/
            retention-days: 7

    Key details per user decisions:
    - Separate job that runs in parallel with lint/typecheck/test (no depends_on/needs)
    - timeout-minutes: 15 for dev server startup + browser tests
    - Uses GitHub Actions secrets for NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY
    - --with-deps flag installs OS-level browser dependencies on ubuntu-latest
    - Upload playwright-report/ as artifact only on failure (if: failure())
    - Uses actions/upload-artifact@v4 (v3 is deprecated)
    - E2E is a required status check -- note to user: must configure in GitHub branch protection settings manually
