1. **Create `src/app/providers.tsx`** ("use client"):
   - Import posthog from "posthog-js", PostHogProvider from "posthog-js/react", usePathname/useSearchParams from "next/navigation", useEffect/Suspense from "react", usePostHog from "posthog-js/react".
   - Initialize posthog in a `typeof window !== "undefined"` guard with:
     - api_host: "https://us.i.posthog.com" (US cloud per user decision)
     - ui_host: "https://us.posthog.com"
     - capture_pageview: false (manual tracking for App Router per user decision)
     - capture_pageleave: true
     - person_profiles: "always"
     - persistence: "localStorage+cookie"
   - Create `PostHogPageView` component that uses usePathname + useSearchParams + usePostHog. In useEffect, build full URL from window.origin + pathname + searchParams and call `posthog.capture("$pageview", { $current_url: url })`. Dependencies: [pathname, searchParams, posthog].
   - Export `PHProvider` component that wraps children with PostHogProvider (client prop = posthog instance). Inside, render `<Suspense fallback={null}><PostHogPageView /></Suspense>` before {children}. The Suspense wrapper is required because useSearchParams needs it in App Router.
   - Guard initialization: if NEXT_PUBLIC_POSTHOG_KEY is not set, PHProvider should just render {children} without PostHog (graceful no-op for local dev without PostHog key).

2. **Create `src/lib/posthog.ts`** — typed analytics helper:
   ```typescript
   import posthog from "posthog-js";

   export const analytics = {
     modelViewed: (modelId: string, modelName: string) =>
       posthog.capture("model_viewed", { model_id: modelId, model_name: modelName }),
     modelCompared: (modelIds: string[]) =>
       posthog.capture("model_compared", { model_ids: modelIds, count: modelIds.length }),
     listingViewed: (listingId: string, listingName: string) =>
       posthog.capture("listing_viewed", { listing_id: listingId, listing_name: listingName }),
     auctionBid: (auctionId: string, amount: number) =>
       posthog.capture("auction_bid", { auction_id: auctionId, bid_amount: amount }),
     lensSwitched: (fromLens: string, toLens: string) =>
       posthog.capture("lens_switched", { from_lens: fromLens, to_lens: toLens }),
     searchPerformed: (query: string, resultCount: number) =>
       posthog.capture("search_performed", { query_length: query.length, result_count: resultCount }),
   };
   ```
   Use snake_case properties. Use query_length instead of raw query text to avoid PII. All methods are fire-and-forget.
