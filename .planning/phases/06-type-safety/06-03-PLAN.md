Fix pattern for API routes:
```typescript
// BEFORE:
// eslint-disable-next-line @typescript-eslint/no-explicit-any
const { data } = await (supabase as any)
  .from("table_name")
  .select("*")
  .eq("id", id);

// AFTER:
const { data } = await supabase
  .from("table_name")
  .select("*")
  .eq("id", id);
```

For .map() callbacks on query results:
```typescript
// BEFORE:
data.map((item: any) => item.field)

// AFTER (if inference works):
data.map((item) => item.field)

// AFTER (if inference doesn't resolve, e.g. complex joins):
data.map((item: { field: string; nested: { id: string } }) => item.field)
```
